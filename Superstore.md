# Superstore Excel Practice Workbook

This workbook is an Excel exercise file built around the classic Superstore sales dataset. It's set up as a self-paced practice file, where each tab covers a different set of Excel skills using the same real-world data (orders, returns, and people/regions).

## File Structure

**Orders** – The main dataset. Contains order-level details like Order ID, dates, customer info, location, product category, sales, quantity, discount, and profit.

**Returns** – A small lookup table showing which Order IDs were returned.

**People** – Maps each sales region to the person responsible for it.

**Section 1 to Section 12** – Practice sections, each focused on a specific Excel topic. Every section has a set of numbered "Tasks" with a short instruction and a space to work out the answer using the Orders data.

## Topics Covered by Section

1. **Section 1** – Basic arithmetic (SUM, MAX, MIN, SMALL, LARGE, MEDIAN), Transpose vs TRANSPOSE formula, Status Bar quick calculations, YoY growth using POWER.
2. **Section 2** – Formatting cells, wrapping text, Format Painter.
3. **Section 3** – Sorting data (dates, sales, product names), conditional formatting + sorting, multi-level sorting.
4. **Section 4** – Working with dates: durations, DATEDIF, TODAY, DATE/YEAR/MONTH/DAY functions.
5. **Section 5** – Logical functions: IF, AND, OR, IFERROR, and combining them for performance labeling.
6. **Section 6** – Data cleaning: removing duplicates, text functions (UPPER, PROPER, LOWER, TRIM), SEARCH/FIND, Go To Special, Text to Columns, CONCATENATE vs &, Find & Replace.
7. **Section 7** – Pivot tables: building, filtering, grouping by date/quarter/year, handling blanks, custom groups, conditional formatting on pivots.
8. **Section 8** – Lookups: VLOOKUP, IFERROR with lookups, named ranges, HLOOKUP vs VLOOKUP.
9. **Section 9** – Conditional aggregation: COUNTIF, SUMIF, AVERAGEIF, COUNTIFS, SUMIFS.
10. **Section 10** – Conditional formatting: highlighting values, color scales, icon sets.
11. **Section 11** – Combining data across multiple sheets/pivot sources.
12. **Section 12** – Intro to VBA macros: what they are, use cases, recording vs writing code, security, and the Developer tab.

## How to Use This File

Go section by section. Each "Task" cell has a short prompt right next to it, usually with a blank space or column where you'd write the formula or result. The Orders sheet is your main source of data for almost every task, so keep it open alongside whichever section you're working on.

## Data Source

The Orders data is the standard Superstore sample dataset, commonly used for Excel, Tableau, and Power BI practice.
